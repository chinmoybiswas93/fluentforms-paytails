== Description ==

**Fluent Forms Paytails** is a powerful WordPress plugin designed to enhance your Fluent Forms experience by offering in-depth payment analytics and a clear overview of total payments received.

** With Fluent Forms Paytails, you can:**

- View a centralized dashboard displaying total payments received across all Fluent Forms.
- Access detailed analytics for each payment receiver form, including transaction counts and amounts.

**Ideal for:**
- E-commerce site owners using Fluent Forms for payments
- Freelancers tracking client payments
- Non-profits managing donations via forms
- Businesses needing detailed payment insights

== Installation ==

1. Upload the plugin to `/wp-content/plugins/fluent-forms-paytails` or install it via the WordPress Plugin Directory.
2. Activate it from the **Plugins** menu in WordPress.
3. Ensure Fluent Forms is installed and configured with payment gateways.
4. Navigate to **Fluent Forms Paytails** in the WordPress dashboard to view payment analytics.

== Usage ==

1. Go to **Paytails** from your Fluent Forms Menu.
2. View total payments received across all forms.
3. Also, explore detailed stats for individual payment receiver forms.

== Frequently Asked Questions ==

= Does this plugin require Fluent Forms to work? =
Yes, Fluent Forms Paytails is an add-on for Fluent Forms and requires the core Fluent Forms plugin to function.

= Which payment gateways are supported? =
The plugin supports all payment gateways compatible with Fluent Forms, including PayPal, Stripe, and more.

= Is the plugin secure for handling payment data? =
Absolutely. The plugin uses secure data handling practices and complies with WordPress standards.

= Does it work on multisite WordPress installations? =
Yes, Fluent Forms Paytails is compatible with both single-site and multisite WordPress environments.
